# Secure Login System

## Overview
A secure web application developed using Flask and MySQL that allows users to register and log in safely using hashed passwords and session management.

## Features
- User Registration and Login
- Password Hashing using bcrypt
- Input Validation
- SQL Injection Protection
- Session Management
- Logout Feature
- Optional Two-Factor Authentication (2FA)

## Technologies Used
- Python
- Flask
- MySQL
- bcrypt
- HTML/CSS

## Security Measures
- Passwords stored in hashed format
- Parameterized SQL queries to prevent SQL Injection
- Secure session handling for authentication

## Expected Outcome
The system improves user account security by preventing unauthorized access and protecting sensitive login information.

## Future Enhancements
- Email Verification
- Password Reset
- CAPTCHA
- JWT Authentication
- 2FA Integration
